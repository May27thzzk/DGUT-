# 莞工小蟑螂 - 优学院全能助手 (xz-ulearning-helper)

从优学院导出课件题库和训练题库，支持自动刷课，兼容标准优学院和东莞理工学院版本。

## 工作原理

### API 接口分析

优学院的课件学习系统基于前后端分离架构，前端通过 REST API 与后端通信。本工具通过分析浏览器网络请求，找到了关键的数据接口：

```
GET /uaapi/course/stu/{courseId}/directory?classId={classId}
```
获取课程目录，返回所有章节和小节结构。

```
GET /uaapi/wholepage/chapter/stu/{nodeId}
```
获取章节详情，返回所有页面和题目数据。

```
GET /uaapi/questionAnswer/{questionId}?parentId={pageId}
```
获取单题答案，返回标准答案和解析。

### 平台差异

| 平台 | API 地址 | 前端地址 |
|------|----------|----------|
| 东莞理工学院 | api.dgut.edu.cn | ua.dgut.edu.cn |
| 标准优学院 | api.ulearning.cn | ua.ulearning.cn |

两个平台的 API 结构完全相同，仅域名不同，脚本自动适配。

### 认证机制

请求需要携带 `AUTHORIZATION` 请求头，值为登录后获取的 Token。Token 存储在：

- Cookie: `AUTHORIZATION` 或 `token`
- 请求头: `authorization` / `ua-authorization`

Token 有时效性，过期后需重新登录，脚本会自动检测并提示。

### 数据处理流程

```
浏览器获取认证 → 调用目录API获取章节 → 逐章获取页面数据 → 4路并发获取答案 → 生成JSON下载
      ↓                    ↓                    ↓                ↓
  AUTHORIZATION         chapters[].items[]    pages[].coursepageDTOList[]    questions[]
```

## 功能特点

### 题库导出

- **课件题库导出** — 按章节选择性导出，4 路并发获取答案
- **训练题库导出** — 一键导出训练页面题目和标准答案
- **导出格式** — JSON，兼容佛脚、Anki 等刷题工具导入
- **导出进度** — 实时进度条，支持随时取消
- **导出历史** — 记录最近 10 次导出，区分课件/训练来源
- **失败自动重试** — 获取答案失败自动重试一次

### 自动刷课

- **自动播放视频** — 支持自定义倍速（1x~15x），自动静音
- **自动答题** — 支持正确率控制（固定值或区间随机，如 95%~98%）
- **自动翻页** — 可设置页面停留时间和最大重试次数
- **范围控制** — 可指定起始/结束章节
- **答题间隔随机化** — 防止被平台检测为脚本操作
- **运行统计** — 实时显示运行时长、页数、题数、正确率

### 界面设计

- **玻璃拟态 UI** — 半透明毛玻璃效果，现代感设计
- **暗色模式** — 一键切换，含选择器弹窗适配
- **面板可拖动** — 自由移动位置，刷新后自动记忆
- **品牌 Logo** — 内嵌小蟑螂图标
- **导航式菜单** — 首页引导 → 功能子页面

### 稳定性

- **API 自动重试** — 请求失败自动重试 3 次
- **登录过期检测** — 401/403 自动提示刷新页面
- **日志上限** — 防止长时间运行内存泄漏
- **并发导出** — 4 路并发获取答案，大幅提速
- **版本更新提示** — 自动检测新版本并通知

## 安装

### 环境要求

- Chrome / Edge / Firefox 浏览器
- [Tampermonkey](https://www.tampermonkey.net/) 扩展

### 安装步骤

1. 安装 Tampermonkey 浏览器扩展
2. 点击 Tampermonkey 图标 → 添加新脚本
3. 将 `莞工小蟑螂-优学院全能助手.user.js` 的内容全部粘贴进去
4. 保存（Ctrl+S）

或从 [Greasy Fork](https://greasyfork.org/) 安装（如已发布）。

## 使用方法

### 导出课件题库

1. 打开优学院课件学习页面（ua.dgut.edu.cn 或 ua.ulearning.cn）
2. 页面右侧出现「小蟑螂」浮动按钮，点击打开面板
3. 切换到「题库导出」，点击「开始导出课件题库」
4. 弹出章节选择界面，勾选需要导出的章节
5. 点击「开始导出」，等待进度完成
6. JSON 文件自动下载

### 导出训练题库

1. 打开优学院训练答题页面（lms.dgut.edu.cn）
2. 点击浮动按钮打开面板
3. 切换到「题库导出」，点击「开始导出训练题库」
4. 等待导出完成，JSON 文件自动下载

### 自动刷课

1. 打开课件学习页面
2. 切换到「自动刷课」
3. 配置参数（倍速、正确率、停留时间等）
4. 点击「开始自动刷课」
5. 随时可以点击「暂停」停止

### 输出格式

```json
[
  {"题型":"选择题","题干":"题干内容","选项":["选项一","选项二"],"答案":"ABC","解析":""},
  {"题型":"判断题","题干":"题干内容","答案":"正确/错误","解析":""},
  {"题型":"填空题","题干":"题干内容和{答案}","解析":""},
  {"题型":"问答题","题干":"题干内容","答案":"答案内容","解析":""}
]
```

### 导入佛脚刷题

[佛脚](https://www.fojiao.app/) 是一款考前刷题工具，支持导入自定义题库，适合考前集中复习。

**导入步骤：**

1. 用本工具导出题库，得到 JSON 文件
2. 打开 [佛脚](https://www.fojiao.app/) 网站或 App
3. 进入「我的题库」→「导入题库」
4. 选择「JSON 格式导入」
5. 上传导出的 JSON 文件
6. 确认导入，即可开始刷题

**佛脚使用技巧：**

- **错题本** — 做错的题目自动归类，方便二刷
- **收藏夹** — 重点题目可以收藏，考前快速复习
- **随机模式** — 打乱题目顺序，避免记答案位置
- **背题模式** — 直接看答案，适合快速过一遍
- **模拟考试** — 按真实考试节奏限时作答

**建议复习流程：**

```
导出题库 → 导入佛脚 → 第一遍刷题（标记错题） → 二刷错题 → 考前快速过一遍
```

> 提示：建议在考试前 2-3 天导出题库，因为教师可能在考前更新题目。

## 常见问题

**Q: 导出的题目没有答案？**
A: 部分题目可能未被教师设置答案，脚本会自动跳过并在控制台记录。

**Q: 自动刷课正确率怎么设置？**
A: 输入单个数字如 `98` 表示固定 98% 正确率；最低填 `95`、最高填 `98` 表示在 95%~98% 之间随机。

**Q: 导出的 JSON 文件怎么用？**
A: 可以导入佛脚、Anki 等刷题工具，或用 Excel/在线 JSON 工具查看。

**Q: 会不会被检测？**
A: 答题间隔已做随机化处理（±30% 波动），但请合理使用，不要过度依赖。

**Q: Token 过期了怎么办？**
A: 刷新页面重新登录即可，脚本会自动检测并提示。
